## 시작하기

### 시스템 요구사항
- **브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)
- **인터넷**: 초기 로딩 시에만 필요
- **JavaScript**: 활성화 필요
- **쿠키/로컬스토리지**: 활성화 필요

### 접속 방법
1. 웹 브라우저를 실행합니다
2. `index.html` 파일을 브라우저에서 엽니다
3. 또는 배포된 URL로 접속합니다

## 시험 응시하기

### 1단계: 사용자 정보 입력
![메인 화면]

1. **이름 입력**: 본인의 실명을 입력합니다
2. **이메일 입력**: 유효한 이메일 주소를 입력합니다
3. **시험 시작하기** 버튼을 클릭합니다

> 💡 **팁**: 이전에 입력한 정보는 자동으로 저장되어 다음 방문 시 자동 입력됩니다

### 2단계: 시험 진행

#### 화면 구성
```
┌─────────────────────────────────────┐
│  응시자 정보 | 타이머 | 진행률        │
├─────────────┬───────────────────────┤
│             │                       │
│  문제 목록   │     문제 영역          │
│  (1-50)     │                       │
│             │                       │
└─────────────┴───────────────────────┘
```

#### 주요 기능
- **타이머**: 60분 카운트다운 (10분 이하 시 빨간색 경고)
- **문제 네비게이션**: 좌측 번호 클릭으로 문제 이동
- **진행률 표시**: 답변한 문제 수 실시간 표시
- **자동 저장**: 10초마다 답안 자동 저장

#### 문제 유형별 답변 방법

**객관식 (1-40번)**
- 4개의 보기 중 하나를 선택
- 선택한 옵션은 파란색으로 표시
- 다른 옵션 선택 시 자동 변경

**단답식 (41-50번)**
- 텍스트 입력란에 답 작성
- 대소문자 구분 없음
- 키워드만 입력해도 정답 처리

#### 네비게이션
- **이전 문제**: 이전 문제로 이동
- **다음 문제**: 다음 문제로 이동
- **제출하기**: 마지막 문제에서만 표시

### 3단계: 시험 제출

1. 모든 문제를 확인한 후 **제출하기** 버튼 클릭
2. 제출 확인 창에서 답변 현황 확인
   - 답변한 문제 수
   - 미답변 문제 수
3. **제출** 버튼을 클릭하여 최종 제출

> ⚠️ **주의**: 제출 후에는 답안 수정이 불가능합니다

## 결과 확인하기

### 점수 확인
제출 즉시 결과 페이지로 이동하여 다음 정보를 확인할 수 있습니다:

- **총점**: 100점 만점 기준
- **등급**: A(우수), B(양호), C(합격), D(불합격)
- **정답률**: 백분율로 표시
- **소요 시간**: 실제 시험에 사용한 시간

### 상세 결과
각 문제별로 다음 정보를 확인할 수 있습니다:

1. **문제 내용**
2. **본인 답안**
3. **정답**
4. **상세 해설**
5. **정답/오답 여부**

### 결과 활용
- **인쇄하기**: 브라우저 인쇄 기능으로 결과 출력
- **PDF 저장**: 인쇄 창에서 "PDF로 저장" 선택
- **다시 응시**: 새로운 시험 응시 가능

## 자주 묻는 질문

### Q: 시험 중에 브라우저를 새로고침하면 어떻게 되나요?
**A:** 자동 저장 기능으로 답안이 보존됩니다. 마지막으로 본 문제부터 이어서 진행할 수 있습니다.

### Q: 시간이 초과되면 어떻게 되나요?
**A:** 60분이 지나면 자동으로 제출됩니다. 작성한 답안까지만 채점됩니다.

### Q: 답안을 수정할 수 있나요?
**A:** 시험 진행 중에는 자유롭게 수정 가능합니다. 제출 후에는 수정이 불가능합니다.

### Q: 복사/붙여넣기가 안 되는데 왜 그런가요?
**A:** 부정행위 방지를 위해 복사/붙여넣기 기능을 제한했습니다.

### Q: 단답식 문제는 어떻게 채점되나요?
**A:** 키워드 기반으로 자동 채점됩니다. 정확한 답이 아니어도 핵심 키워드가 포함되면 정답 처리됩니다.

### Q: 이전 시험 결과를 다시 볼 수 있나요?
**A:** 네, 결과 페이지는 새로운 시험을 시작하기 전까지 유지됩니다.

## 문제 해결

### 페이지가 로드되지 않아요
1. JavaScript가 활성화되어 있는지 확인
2. 브라우저 캐시 삭제 (Ctrl+F5)
3. 다른 브라우저로 시도

### 타이머가 작동하지 않아요
1. 페이지 새로고침 (F5)
2. 브라우저 콘솔에서 에러 확인
3. 로컬스토리지 초기화

### 답안이 저장되지 않아요
1. 로컬스토리지가 활성화되어 있는지 확인
2. 브라우저 설정에서 쿠키/사이트 데이터 허용
3. 시크릿/프라이빗 모드가 아닌지 확인

### 결과 페이지가 표시되지 않아요
1. 시험을 정상적으로 제출했는지 확인
2. 브라우저 콘솔에서 에러 메시지 확인
3. 메인 페이지에서 다시 시작

|--------|------|

## 접근성 기능

### 시각 장애인을 위한 기능
- 모든 이미지에 대체 텍스트 제공
- 고대비 색상 사용
- 충분한 텍스트 크기

### 키보드 사용자를 위한 기능
- 모든 기능 키보드로 접근 가능
- 명확한 포커스 표시
- Tab 순서 최적화

## 보안 및 개인정보

### 데이터 저장
- 모든 데이터는 로컬 브라우저에만 저장
- 서버로 전송되지 않음
- 브라우저 데이터 삭제 시 모든 정보 제거

### 보안 기능
- 복사/붙여넣기 방지
- 우클릭 제한 (선택적)
- 페이지 이탈 경고

## 시험 팁

### 효율적인 시험 전략
1. **빠른 스캔**: 먼저 전체 문제를 빠르게 훑어보기
2. **쉬운 문제 먼저**: 확실한 답을 아는 문제부터 해결
3. **시간 배분**: 객관식 30분, 단답식 20분, 검토 10분
4. **미답변 체크**: 제출 전 미답변 문제 확인

### 단답식 문제 팁
- 핵심 키워드 위주로 작성
- 너무 길게 쓸 필요 없음
- 확실하지 않아도 관련 내용 작성
